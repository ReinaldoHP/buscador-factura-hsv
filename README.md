# Buscador Factura HSV

Sistema de gestión y búsqueda de facturas para el Hospital San Vicente de Arauca E.S.E.

## Descripción
Aplicación de escritorio desarrollada en Python para facilitar la localización, visualización y validación de soportes de facturas (PDF y ZIP) en repositorios locales.

## Requisitos
- Python 3.8+
- Ver `requirements.txt` para dependencias de Python.

## Ejecución
```bash
python main.py
```
